# Wheels on Fire

A multi-page website for a fictional car enthusiast business called Wheels on Fire. Built as a Web Development 1 college project at Technological University Dublin.

---

## Pages

### Home (`index.html`)
- Hero section with the site name and tagline
- Car of the Week feature (McLaren P1)
- Car collection grid showcasing six iconic cars: Lexus LFA, Lamborghini Aventador, Ferrari 360, Toyota Supra MK4, Nissan GT-R R34, and Porsche 911 Carrera GTS

### About Us (`pages/about.html`)
- Background story of the fictional owner John O'Reilly
- Three sections covering his story, how he turned his passion into a business, and an invitation to join the community

### Events (`pages/event.html`)
- Four upcoming car events displayed as cards
- Each card includes an image, date, location and description
- Events include a Drift Event, Classic Car Show, Jap Car Meet, and German Car Event — all held at Mondello Park

### Contact (`pages/contact.html`)
- Business contact details (address, email, phone)
- Contact form with fields for first name, last name, email, reason for inquiry, and message
- Client-side form validation using JavaScript

---

## Features

- Responsive navigation bar with a hamburger menu for mobile screens
- Active link highlighting on each page
- JavaScript form validation on the contact page
- Google Fonts (Roboto Condensed) throughout
- Font Awesome icons in the footer social links
- Consistent header and footer across all pages
